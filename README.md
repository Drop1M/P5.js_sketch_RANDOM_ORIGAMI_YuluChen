# RANDOM ORIGAMI – p5.js Sketch

## Project Description
This is a random interactive system that simulates origami.

## How to Run
Download or clone this repository

## Interaction
- Visual elements are generated randomly
- Sound is triggered during the interaction

## Tools & Libraries
- p5.js
- p5.sound.js
